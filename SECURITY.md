# Security

## Reporting

This project is not production software yet. Please report security issues privately to the repository owner before publishing details.

## Scope

The current application is a local-first keyboard planning UI with optional GitHub save/load flows. Security-sensitive areas include:

- Karabiner export generation.
- File import/export.
- GitHub OAuth callback handling.
- Local browser storage.

## Supported Versions

Only the `main` branch is actively maintained while the project is pre-release.
